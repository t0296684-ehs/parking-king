# 「找位子」UI/UX 改版設計交接文件

## 專案簡介

「找位子」是一個自用的停車 PWA（手機為主），顯示全台公有停車場**即時剩餘車位**（含一般/身障/婦幼分類）與**充電站**（含 Tesla 超充、華城、EVOASIS 等品牌與費率）。資料來自自架的 Cloudflare Worker API，前端為純 HTML/CSS/JS + Leaflet 地圖，部署於 GitHub Pages。

目前版本功能完整、邏輯經過測試，**本次任務只做視覺與體驗改版，不改資料邏輯**。

## 交付檔案

- `index.html`：版面結構 + 全部 CSS（你的主戰場）
- `app.js`：資料邏輯 + 渲染函式（只能動下列指定函式）
- `manifest.json`、`sw.js`、`icon-192.png`、`icon-512.png`：不需要動（圖示想重畫可以，維持 192/512 尺寸與檔名）

## 使用情境（設計依據）

- 開車前或路邊臨停時單手操作，畫面資訊要一眼可讀
- 常在車內使用，白天有陽光眩光：對比度要足、字級不能小
- 使用者是 Tesla 車主：Tesla 相關元素值得被突顯
- 核心決策只有一個：「哪裡有位子 / 哪裡能充電 → 導航過去」，其他都是次要資訊

## 可以改的範圍

1. `index.html` 內全部 CSS 與版面結構
2. `app.js` 內**僅限**這些渲染函式的 HTML 模板與樣式 class：
   - `cardHtml()` — 清單卡片
   - `openDetail()` — 詳情面板
   - `pinHtml()` — 地圖圖釘
   - `renderBrandChips()` — 品牌篩選鈕
3. 字體、配色、動效、圖示風格全部開放重新設計

## 不可以改的（改了會壞）

1. `app.js` 中渲染函式以外的任何程式碼（資料載入、快取、定位、篩選邏輯）
2. JS 依賴的元素 id / class / data 屬性（可加新的、不可改名或移除）：
   - id：`map`、`list`、`detail`、`scrim`、`banner`、`updated`、`citySel`、`brandChips`、`refreshBtn`、`locateBtn`、`detailNav`
   - class + data：`.chip[data-filter]`（值固定 all/ev/open）、`.bchip[data-brand]`、`.card[data-i]`、`.navbtn[data-i]`
3. 圖釘 class 語意：`.pin.high / .mid / .low / .none / .ev`（Leaflet divIcon 使用，尺寸錨點在 JS 內為 34×34）
4. `fetch`、`localStorage`、Service Worker 註冊等行為
5. 外部依賴僅 Leaflet 1.9.4（CDN）；不可引入需要編譯的框架（無 build step，檔案直接部署）

## 必須保留的資訊語意

- 剩餘車位顏色語意：綠 = 充足（>20）、黃 = 不多（5~20）、紅 = 將滿（<5）、灰 = 無即時資料；色值可調整但四級語意與可辨識度必須保留
- 停車卡片：主徽章顯示「一般剩 N 位」；有分類時顯示「一般 47｜身障 2｜婦幼 1」明細——這是本 App 的核心賣點，視覺上要清楚
- 充電卡片：品牌標籤（Tesla 需突顯）+ 費率原文（可能很長，需處理截斷與展開）
- 每張卡片與詳情面板都要有明顯的「導航」行動按鈕

## API 資料形狀（供設計時參考真實內容）

停車場（`/api/parking?city=Taoyuan`）單筆：

```json
{
  "id": "AO070012", "name": "大湳公有停車場", "lat": 24.95, "lon": 121.22,
  "fare": "平日計時每小時30元，當日最高100元…（可能很長）",
  "address": "桃園市…", "ev": true,
  "avail": 50, "total": 170,
  "spaces": [
    { "t": 1, "n": 165, "a": 47 },
    { "t": 4, "n": 3, "a": 2 },
    { "t": 5, "n": 2, "a": 1 }
  ],
  "time": "2026-07-06T10:58:00+08:00"
}
```

（`t` 為車位類型：1 一般、4 身障、5 婦幼；`n` 總數、`a` 剩餘；`avail` 可能為 null = 無即時資料）

充電站（`/api/charging?city=Taoyuan`）單筆：

```json
{
  "id": "24211954-S3380001", "name": "台茂購物中心立體停車場",
  "lat": 25.05302, "lon": 121.28763,
  "brands": ["華城EValue", "Tesla"], "points": 1, "spaces": 26,
  "chargingRate": "華城：慢充8.3元/度、快充9.8元/度；特斯拉超充：0~60kW:5元/分…（可能更長）",
  "parkingRate": "計時40元/小時", "serviceTime": "全天開放",
  "floors": "1F、6A", "address": "桃園市蘆竹區南崁路一段112號",
  "desc": "立體停車場6A：華城電能-通用型充電樁；立體停車場1F：特斯拉超級充電站"
}
```

## 現有畫面結構（由上而下）

1. Header：Logo「P 找位子」+ 縣市下拉（citySel）+ 更新時間 + 定位鈕 + 重新整理鈕
2. 篩選列：全部 / ⚡充電 / 只看有位
3. 品牌列（僅 ⚡ 模式顯示，橫向捲動）：全部品牌 / Tesla / 華城EValue / …
4. 錯誤橫幅（banner，平常隱藏）
5. 地圖（46vh，Leaflet + OSM，圖釘顯示剩餘數字並變色）
6. 清單（剩餘空間捲動，依距離排序，最多 40 張卡片）
7. 詳情面板（底部抽屜，蓋在 scrim 上）
